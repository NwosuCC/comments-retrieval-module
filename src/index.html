Quickly download all customers' reviews

Enter/paste a supported product URL and retrieve all comments and reviews on the product review page. The retrieved comments and reviews are properly formatted and bundled into a CSV file ready to be delivered.

Of course, you can choose how the output file is delivered to you:

1.) Instant download

This is the default option.

2.) File sent to email address

This option is automatically used when data retrieval has lasted more than five (5) minutes, even if "Instant download" (option 1.) has been selected
